モータ補正係数計算

事前準備
1. main.cpp を開く
2. #define PWM_RESEARCH を有効にする(有効になっていることを確認する)
3. USER_GYRO_OFFSET の値を nxtOSEK/samples_c/helloworld を用いて適宜調整する。
4. make all し、プログラムをNXT本体にアップロードする。

ログ取得
1. NXT本体を机の上に寝かせ、プログラムを実行する。
2. ログを取得するため、Bluetooth接続をする。
3. タッチセンサボタンを押し、走らせる。
4. 徐々にスピードが挙がる動きをする。終わるまで待つ。
ログにPWM値指定時点における左右モータエンコーダ値変化量(速度)が記録される。

モータ補正係数解析
左速度 / 右速度をプロットしてみる。眺めてみる。
左速度 - 右速度をプロットしてみる。眺めてみる。

計算

次の式で算出できそう？
    if (左速度 / 右速度 > 1.0) {
        左係数 = 右速度 / 左速度;
        右係数 = 1.0;
    }
    else {
        左係数 = 1.0;
        右係数 = 左速度 / 右速度;
    }

検証
1. main.cpp を開く
2. #define PWM_RESEARCH をコメントアウトし、#define PWM_VERIFICATION を有効にする。
3. 計算した係数を float PWM_L および float PWM_R にセットする。
4. make all し、プログラムをNXT本体にアップロードする。
5. 走らせてみる。補正前より真っ直ぐ走れば成功。
